本项目为基于卷积神经网络的MPU6050传感器标准库手势识别，可以根据手势识别出敲门、打开和关闭的动作并显示在OLED显示屏上。后续可通过接入其他外设实现开关灯、开关电机等操作。
本文档为I2C采集MPU6050六轴数据并将数据通过SPI存储到W25Q64中。
保存后的数据可以传输到计算机，经过数据处理后可以直接作为卷积神经网络的输入。
其中，I2C选用stm32f108的内置外设，即采用硬件控制，接入PB10和PB11，采用I2C2.
传感器数据通过SPI协议记录到W25Q64中。
为了消除静止状态下的冗余数据，采取按钮控制数据记录的操作，当按下按钮时，将采集到的数据记录在W25Q64中，OLED显示当前为阅读状态。当再按下按钮时，数据记录停止。
